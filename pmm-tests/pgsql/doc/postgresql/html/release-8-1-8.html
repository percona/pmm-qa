E.238.1. Migration to Version 8.1.8

A dump/restore is not required for those running 8.1.X. However, if you are upgrading from a version earlier than 8.1.2, see Section E.244.

  • Remove overly-restrictive check for type length in constraints and functional indexes(Tom)
