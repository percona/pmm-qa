E.219. Release 8.2.3

E.219.1. Migration to Version 8.2.3
E.219.2. Changes

Release date: 2007-02-07

This release contains two fixes from 8.2.2. For information about new features in the 8.2 major release, see Section E.222.

E.219.1. Migration to Version 8.2.3

A dump/restore is not required for those running 8.2.X.

E.219.2. Changes

  • Remove overly-restrictive check for type length in constraints and functional indexes(Tom)

  • Fix optimization so MIN/MAX in subqueries can again use indexes (Tom)
